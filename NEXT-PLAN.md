## e0.3-beta 合格条件（止まらず最後まで）

```
住宅屋根伏図(PDF) → 縮尺較正 → 屋根A → 屋根B → 下屋 → Measurement保存 → measurements.json → 終了
```

- [ ] PDF読込
- [ ] 縮尺較正（実長mm → px/m）
- [ ] 屋根A / 屋根B / 下屋 を Polygon で拾う
- [ ] Measurement 保存
- [ ] measurements.json 出力

これが**一度も詰まらず**終われば、`e0.3-geometry-editor` へ昇格。
